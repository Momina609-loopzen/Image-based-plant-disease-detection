# Image-Based Plant Disease Detection System

## Overview

The **Image-Based Plant Disease Detection System** is a deep learning project developed using **Python**, **TensorFlow**, and **Keras**. The project aims to classify plant leaf images as **Healthy** or **Diseased** using a Convolutional Neural Network (CNN). It demonstrates how Artificial Intelligence can assist in the early detection of plant diseases and support smarter agricultural practices.

## Objectives

* Build a CNN model for binary image classification.
* Preprocess plant leaf images before training.
* Train and evaluate the model on sample data.
* Understand the application of deep learning in agriculture.

## Project Workflow

The system follows a straightforward image classification pipeline:

1. Collect and prepare the plant leaf images.
2. Preprocess the dataset for training.
3. Train the CNN model.
4. Evaluate the model using test images.
5. Predict whether the input leaf is **Healthy** or **Diseased**.

## Model Architecture

A **Convolutional Neural Network (CNN)** was used as the core model for image classification. The network learns visual features from plant leaf images during training and uses those learned patterns to classify new images into one of the two categories.

* **Development Environment:** Google Colab (Jupyter Notebook)
* **Frameworks:** TensorFlow, Keras
* **Libraries:** NumPy, Matplotlib
