# Image-Based Plant Disease Detection System

## Overview

The **Image-Based Plant Disease Detection System** is a deep learning project developed using **Python**, **TensorFlow**, and **Keras**. The project aims to classify plant leaf images as **Healthy** or **Diseased** using a Convolutional Neural Network (CNN). It demonstrates how Artificial Intelligence and computer vision can be applied to support the early detection of plant diseases in agriculture.

## Objectives

* Build a CNN model for binary image classification.
* Preprocess plant leaf images before training.
* Train and evaluate the model using sample data.
* Explore the application of deep learning in agriculture.

## Project Workflow

The system follows a simple image classification pipeline:

1. Collect and prepare plant leaf images.
2. Preprocess the dataset for training.
3. Train the CNN model.
4. Evaluate the model using test images.
5. Predict whether the input leaf is **Healthy** or **Diseased**.

## Model Architecture

A **Convolutional Neural Network (CNN)** was used as the core model for image classification. During training, the network automatically learns important visual features from plant leaf images and uses them to classify new images into the **Healthy** or **Diseased** category. This approach makes CNNs well-suited for image recognition tasks without requiring manual feature extraction.

## Technologies Used

* **Programming Language:** Python
* **Development Environment:** Google Colab (Jupyter Notebook)
* **Frameworks:** TensorFlow, Keras
* **Libraries:** NumPy, Matplotlib

## Conclusion

This project provided practical experience with image preprocessing, CNN-based image classification, and model evaluation using TensorFlow and Keras. It also demonstrated how deep learning techniques can be applied to solve real-world agricultural problems through automated plant disease detection.
